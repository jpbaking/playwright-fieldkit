# Playwright FieldKit intent router

When the user asks to explore, test, debug, audit, compare, record, demonstrate,
plan test data/matrices, analyze coverage, generate negative paths, create a test
charter, triage a flaky test, or understand a live website, activate the
`playwright-fieldkit` skill. If
automatic skill activation is unavailable or missed, read and follow
`.cline/skills/playwright-fieldkit/SKILL.md` directly.

Apply the matching workflow immediately; do not merely tell the user which slash
command they could run. Shortcut workflows remain available under
`.clinerules/workflows/`.

Only operate on authorized targets. Never add `--i-am-authorized` unless the user
explicitly confirms authorization.
